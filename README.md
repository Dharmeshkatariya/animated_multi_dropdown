## ✨ Features

- 🎨 **25+ Animation Types**: Glass morphism, Liquid wave, Neon glow, Cyberpunk, Hologram, and many more
- 🎯 **Single & Multiple Selection**: Support for both selection modes
- 🔍 **Built-in Search**: Optional search functionality with customizable styling
- 🏷️ **Chip Display**: Display selected items as beautiful chips in multi-select mode
- 🖱️ **Haptic Feedback**: Optional vibration feedback on interactions
- 📱 **Fully Responsive**: Works seamlessly on mobile, tablet, and web
- 🎨 **Extremely Customizable**: 50+ configuration options for complete control
- 🚀 **Performance Optimized**: Smooth 60fps animations
- 🎯 **Type Safe**: Fully generic with complete type safety
- 🌙 **Dark Mode Ready**: Works with any theme
- ♿ **Accessibility**: Support for screen readers and keyboard navigation
